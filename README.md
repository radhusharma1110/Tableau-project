# Sales Performance Analysis – Supermarket Dataset

## Project Overview
This project analyzes the **Sample Supermarket Dataset** from Kaggle to uncover sales trends,
profit patterns, and insights using **Tableau**.

## Objectives
- Analyze overall sales & profit
- Identify peak seasons
- Understand performance by region & product category
- Provide visual dashboards for business insights

## Tools
- Tableau (Public or Desktop)
- CSV dataset

## Dashboards Included
- Sales Trend Over Time
- Profit Trend Analysis
- Regional Sales Map
- Category Performance
- Top 10 Products by Sales

## Key Insights
- West region contributes highest revenue
- Technology has highest sales
